# pokereact

---

App hosted on [noxsios.net](http://noxsios.net)

Site made with:

- React
- React Material UI
- Pokedex Promise v2

---

To self host :

1. Git clone `https://github.com/Noxsios/pokereact`
2. cd `pokereact`
3. `npm install`
4. `npm start`
5. [Create React App](https://github.com/facebook/create-react-app) tools
